# SafeUni Lima – Landing Page

Landing page informativa para el proyecto SafeUni Lima, una solución de seguridad universitaria que conecta a estudiantes y autoridades ante situaciones de emergencia.

## Propósito
Brindar información clara sobre cómo SafeUni ayuda a los estudiantes a pedir ayuda rápida y a las autoridades a atender alertas en tiempo real.

## Público objetivo
- Estudiantes universitarios
- Serenazgos y policías municipales
- Áreas de seguridad universitaria

## Características de la Landing
- Presentación de beneficios según rol
- Explicación del funcionamiento del sistema SOS
- Llamado a acción para estudiantes y autoridades
- Formulario de contacto

## Tecnologías
- HTML5
- CSS3
- JavaScript
